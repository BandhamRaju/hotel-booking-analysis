# 📊 Hotel Booking Analysis

## 📌 Project Overview
This project analyzes hotel booking data using SQL and Power BI to uncover business insights related to booking trends, cancellations, revenue performance, and customer behavior.

---

## 🔧 Tools Used
- SQL (PostgreSQL)
- Power BI
- Data Cleaning
- Data Transformation

---

## 📈 Key Metrics
- Total Bookings: 119K
- Average Daily Rate (ADR): $101.83
- Cancellation Rate: 37.04%
- Revenue: $42.72M

---

## 🔍 Key Insights
- High cancellation rate (~37%) indicates potential pricing or operational issues  
- City hotels have higher bookings but also higher cancellations  
- Resort hotels show more stable and reliable bookings  
- Booking demand peaks during July and August (seasonality)  
- Longer lead time results in higher cancellations  
- Transient customers contribute most bookings but cancel more frequently  
- Online Travel Agencies generate the highest revenue  
- More special requests reduce cancellation probability
